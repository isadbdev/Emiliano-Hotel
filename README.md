# Emiliano Hotel

Projeto de site fictício de hotel desenvolvido com HTML, CSS, JavaScript e Bootstrap.

## Funcionalidades

- Página inicial responsiva
- Seção de quartos
- Formulário de reserva
- Layout adaptado para mobile e desktop

## Tecnologias usadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Acesso

O site pode ser acessado via GitHub Pages após publicação.
